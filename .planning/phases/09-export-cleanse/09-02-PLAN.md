## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user→date input | cleanseDate from `<input type="date">` (browser-validated YYYY-MM-DD) feeds deriveCleanseCount/filterCleansed |
| dashboard page→chrome.storage.local | cleanse write replaces flaggedAccounts/storedPosts and wipes dismissedAccounts |
| chrome.storage.local→download file | export writes stored post text (JSON only) to the user's filesystem |

| T-09-04 | Tampering | handleClean fired with no date | mitigate | Confirm cleanse button `disabled={!cleanseDate}` plus early-return guard `if (!cleanseDate || !cleansePreview) return` (Pitfall 6). |
