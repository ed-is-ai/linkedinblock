<objective>
Add a "Profile bot rate" stat to the dashboard. This answers a different question from the existing post-rate: of all *unique people* seen in the feed during the time window, what percentage were flagged bots?

The computation:
1. Filter `dailyStats` to the selected window (same `cutoff` logic as existing `windowStats`).
2. Union all `seenProfileIds` arrays from those entries into a single Set → `seenUnion`.
3. Count flagged, non-dismissed accounts whose `authorId` appears in `seenUnion` → `botProfileCount`.
4. Rate = `botProfileCount / seenUnion.size × 100`.

Display: add a new stat entry to the existing "Posts hidden — all time" card, or as a sibling card. The spec says "visible on the dashboard without requiring any popup changes" — a new sub-row in the existing top card is the minimal change.
</objective>

    ```typescript
    // Profile bot rate — unique profiles seen in window that are flagged (INSIGHT-02)
    const seenUnion = new Set<string>();
    for (const d of windowStats) {
      for (const id of (d.seenProfileIds ?? [])) {
        seenUnion.add(id);
      }
    }
    const pendingIds = new Set(
      accounts
        .filter(a => a.status !== 'dismissed' && !dismissed.includes(a.authorId))
        .map(a => a.authorId)
    );
    const botProfileCount = [...seenUnion].filter(id => pendingIds.has(id)).length;
    const profileBotRate = seenUnion.size > 0
      ? ((botProfileCount / seenUnion.size) * 100).toFixed(1)
      : null;
    ```
