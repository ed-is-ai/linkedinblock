## Key Design Decisions

| Decision | Outcome |
|----------|---------|
| Icon location | `src/public/icons/` (Vite publicDir) — copied verbatim to `dist/icons/` |
| Icon generation | SVG source + `sharp` devDep in `scripts/generate-icons.js` |
| Privacy policy hosting | `PRIVACY.md` in repo root, served as raw GitHub URL |
| `api.anthropic.com` permission | Kept in `host_permissions`; documented as opt-in in store listing |
| Packaging | `archiver` devDep; `npm run package` = build + zip |
| Version | `1.2.0` in both `manifest.json` and `package.json` |
