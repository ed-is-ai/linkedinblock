<objective>
Create the extension's branded icon at 16, 48, and 128px.

Design: LinkedIn blue (`#0a66c2`) rounded square, white shield with a blue diagonal slash through it — visually communicates "content filtering/blocking" while matching LinkedIn's brand colour.

Pipeline:
1. `src/icons/icon.svg` — SVG source, 128×128 viewBox
2. `scripts/generate-icons.js` — Node.js ESM script using `sharp` to rasterise at three sizes
3. Run the script to produce the three PNG files and commit them

`sharp` is added as a devDependency. The generated PNGs are committed to the repo (not regenerated at build time) so they're available to `vite build` without running the script.
</objective>

    **1. Create `scripts/generate-icons.js`:**

    ```javascript
    #!/usr/bin/env node
    // Generates PNG icons from src/icons/icon.svg at 16, 48, 128px.
    // Run once: node scripts/generate-icons.js
    import sharp from 'sharp';
    import { readFileSync } from 'fs';
    import { dirname, join } from 'path';
    import { fileURLToPath } from 'url';

    const __dirname = dirname(fileURLToPath(import.meta.url));
    const svgPath = join(__dirname, '../src/icons/icon.svg');
    const svgBuffer = readFileSync(svgPath);

    const sizes = [16, 48, 128];
    for (const size of sizes) {
      const outPath = join(__dirname, `../src/icons/icon-${size}.png`);
      await sharp(svgBuffer).resize(size, size).png().toFile(outPath);
      console.log(`Generated icon-${size}.png`);
    }
    console.log('All icons generated.');
    ```

    **2. Add `sharp` to devDependencies in `package.json`:**

    In the `devDependencies` section, add:
    ```json
    "sharp": "^0.33.0"
    ```

    **3. Add a `generate-icons` script to the `scripts` section of `package.json`:**
    ```json
    "generate-icons": "node scripts/generate-icons.js"
    ```
  </action>
