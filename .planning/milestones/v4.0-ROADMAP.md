## Changes

**`src/background/index.ts`** only:
1. `SYSTEM_PROMPT` expanded from ~450 to 856 words — adds `impersonal-framing` and `no-specificity` signals, scoring calibration table, hard rules (short posts, typos, non-English)
2. `'anthropic-beta': 'prompt-caching-2024-07-31'` added to fetch headers
3. `system: SYSTEM_PROMPT` (string) → `system: [{ type: 'text' as const, text: SYSTEM_PROMPT, cache_control: { type: 'ephemeral' as const } }]`

## Key Design Decisions

| Decision | Outcome |
|----------|---------|
| Cache scope | System prompt only — user message (post text) is always unique, not cacheable |
| Cache TTL | 5 minutes (Anthropic ephemeral default) |
| Token minimum | Sonnet requires ≥1024 tokens; prompt expanded to 856 words (~1113 tokens) |
| `as const` on type literals | Required to keep TypeScript literal types narrow enough to satisfy the API shape |
