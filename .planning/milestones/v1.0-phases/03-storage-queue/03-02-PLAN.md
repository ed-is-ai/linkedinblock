    Create a new file `src/content/detector/signals/profile.ts`. This module must:

    **1. Headline formula signal constants:**

    Export `HEADLINE_BUZZWORDS: readonly string[]` — an array of lowercase strings that indicate a formulaic AI-generated headline. Include at minimum:
    `['visionary', 'guru', 'ninja', 'evangelist', 'thought leader', 'disruptor', 'change maker', 'innovator', 'futurist', 'influencer', 'helping you', 'helping others', 'empowering', '| speaker', '| coach', '| mentor', '| advisor', '| consultant', '| trainer', '| author', '| entrepreneur']`

    Export `HEADLINE_PIPE_PATTERN: RegExp` — matches pipe-separated role combos: `/(?:\w[\w\s]+\|\s*){2,}/`. Two or more pipe-separated segments is the formula pattern.

    **2. checkHeadlineFormula(headlineText: string): number**
    Pure function. Receives the raw headline text. Returns a score 0–10:
    - If `HEADLINE_PIPE_PATTERN` matches: +5 pts
    - For each buzzword in `HEADLINE_BUZZWORDS` that appears (case-insensitive) in the text: +2 pts per hit, capped at +5 pts total from buzzwords
    - Maximum total: 10 pts (clamp with Math.min)
    - If headlineText is empty/blank: return 0

    **3. checkConnectionDegree(degreeText: string): number**
    Pure function. Receives the text content of the degree indicator element. Returns:
    - `5` if the normalised text contains `'3rd'` or `'3rd+'`
    - `0` for `'1st'`, `'2nd'`, or anything else
    Signal key: `'degree-3'`

    **4. extractProfileSignals(postNode: Element): Record<string, number>**
    DOM extraction helper. Queries `postNode` using the two selectors from selectors.ts and calls the above signal functions:

    ```
    import { AUTHOR_HEADLINE, CONNECTION_DEGREE } from '../../selectors';

    export function extractProfileSignals(postNode: Element): Record<string, number> {
      const signals: Record<string, number> = {};

      const headlineEl = postNode.querySelector(AUTHOR_HEADLINE);
      const headlineText = (headlineEl as HTMLElement | null)?.innerText?.trim() ?? '';
      const headlineScore = checkHeadlineFormula(headlineText);
      if (headlineScore > 0) signals['headline-formula'] = headlineScore;

      const degreeEl = postNode.querySelector(CONNECTION_DEGREE);
      const degreeText = (degreeEl as HTMLElement | null)?.innerText?.trim() ??
                         (degreeEl?.getAttribute('aria-label') ?? '');
      const degreeScore = checkConnectionDegree(degreeText);
      if (degreeScore > 0) signals['degree-3'] = degreeScore;

      return signals;
    }
    ```

    Use `innerText` (not `textContent`) for visible text extraction, consistent with the rest of the codebase. Fall back to `getAttribute('aria-label')` for the degree element since some degree indicators are aria-labelled rather than text nodes.

    **5. Signal key naming:** Use `'headline-formula'` and `'degree-3'` as the Record keys, matching D-09 in 03-CONTEXT.md.

    **6. No default export** — all exports are named. Follow the same module style as other signal files.
